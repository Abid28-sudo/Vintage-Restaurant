# Vintage Restaurant

A clean, responsive portfolio website for a vintage restaurant created with HTML, CSS, and JavaScript.

## Features

- Multi-page layout: Home, Menu, Offers, About
- Responsive design for mobile and desktop
- Simple navigation and visual focus on classic dining ambiance
- Easy-to-edit sections for restaurant details, menu items, and promotions

## Usage

1. Open `home.html` in your browser.
2. Navigate to other sections via the menu.
3. Customize content in `index.html`, `about.html`, `menu.html`, `offers.html`, `style.css`, and `script.js`.

## Tech Stack

- HTML
- CSS
- JavaScript
